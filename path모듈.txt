03/path.js
절대 경로: 파일이나 폴더의 위치를 루트root 폴더더부터 시작해서 파일까지 전부 다 나타내는 것
상대경로: 기준이 되는 파일이 있을 경우 사용하는 경로
         같은 폴더에 있을 경우 "./"로 표기 한단계 위 부모 폴더에 있을 경우 "../" 사용

__dirname: 현재 실행중인 파일의 디렉토리 절대 경로 반환
__filename: 현재 실행 중인 파일의 디렉토리 경로 + 파일이름 반환

join: 문자열이 하나로 묶여서 경로가 만들어짐
dirname: 디렉토리 경로만 반환, 입력된 경로의 부모 경로 출력
basename: 입력된 경로의 파일 이름만 추출
