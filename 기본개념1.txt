정보를 요청하는 컴퓨터: 클라이언트
정보를 처리하는 컴퓨터: 서버


서버를 만든다 -> 서버에서 실행할 프로그램을 만든다


Node.js: 언어가 아닌 자바스크립트라는 언어로 서버를 개발할수 있도록 도와주는 도구


Express 프레임워크: 서버를 만들 때 마다 반복되는 복잡한 기능을 처리해주는 함수를 제공


모듈: 기능별로 쪼개놓은 파일
CommonJS 모듈 시스템: Node.js의 기본 모듈 시스템(require, module.exports)
ES 모듈 시스템: ECMAScipts의 표준 모듈 시스템(import, export)


라이브러리: 여러개의 모듈을 묶어서 하나의 라이브러리로 만듦
패키지: 배포할 수 있도록 여러 모듈과 모듈 관련 파일을 묶어 놓은 것



npm: Node Package Manger
자바스크립트 라이브러리를 비롯해  프레임워크, 플러그인 등 다양한 패키지를 내려받아 사용할 수 있게 해주는 도구

npm init -> pacakage.json파일에 어떤 패키지들을 가져다 사용했느지의 정보가 담김
